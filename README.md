# Infinix Landing Page

**Premium showcase for Infinix mobile devices and accessories.**

---

## 📋 Overview

Clean, responsive landing page highlighting Infinix's latest smartphones, wireless earbuds, power banks, chargers, smart TVs, tablets, and more.

Designed to convert visitors into customers with beautiful product imagery and clear calls-to-action.

---

## ✨ Features

- Hero banner with key products
- Product categories grid (Smartphones, Audio, Power, TVs, etc.)
- Responsive design for all devices
- Smooth animations and modern UI
- Fast-loading static site

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3** (Custom responsive styling)
- **Vanilla JavaScript**
